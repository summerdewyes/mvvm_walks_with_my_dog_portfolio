어야가자 (Walk-With-My-Dog)
==
[기존 어야가자](https://github.com/summerdewyes/walks-with-my-dog-portfolio)에 MVVM 패턴을 적용했습니다.   
반려견과 함께한 산책을 기록합니다.
   
   
## 사용 라이브러리

* Google Map API
* Jetpack
* Coroutines
* Dagger
* Retrofit2
* Glide
* CardView
